### FONTS:

**Reference:** font-family:'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
This Specifies a prioritized list of font family names or generic family names.The browser iterates through the list of family names until it matches a font that is available in the system.
Segoe UI --> Main Font (priority)
Tahoma, Geneva, Verdana, sans-serif -->These are the Fallback Fonts
